# Movies Server API

API REST para gestionar películas, construida con **Express.js** y **MongoDB (Mongoose)**.

---

## Tecnologías utilizadas

- Node.js
- Express.js
- MongoDB
- Mongoose
- Nodemon (desarrollo)
